# 에이전트에 툴을 몇개까지 줄까

## AI에게 직접 물어보자

GPT, 클로드, 제미나이는 모두 5~10개 이내를 권장하며 모델 성능이 높을수록 조금 더 늘릴 수 있다고 대답했다. 그런데 툴이 비슷한 종류가 많아서 헷갈릴수는 있어도 명확히 구분되는 툴인데 5~10개는 너무 적은거 같다.

## 실제로도 그런지 실험을 해보자

단순 숫자, 문자열을 연산하는 툴 30개 만들고 의도한 툴을 사용하는지 테스트해 보았다. 결과는 의도한대로 올바른 툴을 사용했다. 그러나 툴 자체가 너무 간단한 기능이라는점 그리고 모델을 GPT-4o mini라는 나름 준수한 성능의 모델이라는것을 당연히? 처리할 수 있는 수준이라 생각한다.

## 결론은

명확한 갯수로 제한하기 보다는 실제로 테스트 해보는게 최선인것 같다. 지금 진행중인 프로젝트에서도 에이전트가 이상한 툴을 선택하지 않도록 최대한 툴을 줄일려고 하는데 테스트 해보면서 수정해야겠다.
