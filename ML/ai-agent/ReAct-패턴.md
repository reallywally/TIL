# ReAct 패턴

## ReAct 패턴이란

2022년 Google Brain 연구팀이 발표한 논문에서 제안된 개념으로, 기존의 단순 LLM 호출보다 훨씬 복잡한 작업을 처리할 수 있습니다.
핵심은  Thought(추론) -> Action(행동) -> Observation(관찰)의 순환 루프 전체 흐름
